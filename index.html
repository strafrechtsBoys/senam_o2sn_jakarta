👥 Data Atlet

Kelola identitas peserta: nama, sekolah, wilayah asal, dan kategori. Nilai diatur terpisah di menu Input Nilai.

Tambah Atlet Baru

Data nilai default 0.000 saat baru ditambahkan.

Catatan: setelah atlet terdaftar, isi nilai di menu Input Nilai. Identitas atlet dapat diedit kapan saja tanpa menghilangkan nilai yang sudah ada.

📊 Import / Export Massal

Untuk input banyak atlet sekaligus, gunakan template Excel.

Cara pakai: klik Download Template, isi data di Excel (jangan ubah header), simpan, lalu klik Import dari Excel. Atlet baru akan ditambahkan; atlet dengan nama+sekolah+kategori sama akan di-update.

🖨️ Print & Export

Cetak rekap nilai untuk publikasi atau unduh data dalam format yang bisa dibuka Excel.

Cetak Rekap Nilai

Membuka rekap rapi di jendela baru dan langsung memunculkan dialog cetak. Halaman admin tetap pada posisinya.

Format A4 landscape, otomatis terbagi per kategori Putra dan Putri, urut nilai tertinggi. Pastikan pop-up diizinkan oleh browser.

Export Data Nilai

Unduh seluruh data atlet beserta nilai akhir. Format Excel direkomendasikan untuk pengolahan lanjut.

Format Excel (.xlsx) lebih baik untuk arsip — kolom otomatis ter-format dan terlebar sesuai isi. CSV cocok kalau perlu di-import ke sistem lain.
