🖨️ Print & Export

Cetak rekap nilai untuk publikasi atau unduh data dalam format yang bisa dibuka Excel.

Cetak Rekap Publik

Membuka tampilan halaman publik dan langsung memunculkan dialog cetak.

Export Excel (CSV)

Unduh seluruh data atlet beserta nilai dalam format CSV. Dapat dibuka langsung di Microsoft Excel atau Google Sheets.

File CSV menggunakan UTF-8 BOM agar karakter Indonesia tampil benar di Excel.
